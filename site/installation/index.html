Installation¶

Stable release¶

To install urbanity, run this command in your terminal:

pip install urbanity

This is the preferred method to install urbanity, as it will always install the most recent stable release.

If you don't have pip installed, this Python installation guide can guide you through the process.

From sources¶

The sources for urbanity can be downloaded from the Github repo.

You can clone the public repository:

git clone git://github.com/winstonyym/urbanity
